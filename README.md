# README

## Overview
This repository contains the **code and data** used for the modeling section presented in the paper:  
**“Mapping DNA sequence logic of base-excision repair enzyme binding reveals mechanistic principles and links to genome evolution”** *(title placeholder — update to match the final manuscript title).*

The files here reproduce the computational framework described in the **Specificity defined by base identity and dependencies** sections of the paper, which quantify how individual and cooperative effects of flanking sequences influence enzyme–DNA recognition.

---

## Contents
- **`Jupyter notebooks for TDG and UDG`** – notebooks for additive and interaction-based modeling of enzyme binding.  
- **`data/`** – Processed binding data (as described in Method section in the paper) used as input for model training and validation.  

---

## Usage
1. Clone or download the repository.
2. Install dependencies (Python ≥3.7).
3. Run each notebook independently, figures will be displayed accordingly.
